Overview

LiquidFun is a 2D rigid body and fluid simulation C++ library for games based upon Box2D. Programmers can use it in their games to make objects move in realistic ways and game worlds more interactive. From the game engine's point of view, a physics engine is just a system for procedural animation.

Releases

Stable releases of LiquidFun are available for download from github.com/google/liquidfun/releases.

LiquidFun source code is available for download from github.com/google/liquidfun.

Documentation

LiquidFun's documentation consists of Build Instructions, a Programmer's Guide and API reference.

The Build Instructions describe how to build the library, examples and unit test binaries from source for each target operating system. Start working through the Build Instructions after downloading the library's source.

A background of game physics, descriptions of LiquidFun's component and examples of how to use the library are provided in the Programmer's Guide.

The API reference is intended to be used by advanced developers who have already read the Programer's Guide since it provides detailed descriptions of all C++ objects and functions in the library.

Discussions with LiquidFun users and developers occur on the mailing list. Issues with the LiquidFun library can be filed on the issues tracker.
