float32 b2ParticleGroup::GetAngle ( ) const

Get the rotational angle of the particle group as a whole. Used only with groups of rigid particles.

uint32 * b2ParticleGroup::GetParticleFlagsBuffer ( )

Gets an indexed buffer containing the type of each particle in the group. The indices in the buffer match those of the particles themselves. For instance, the behavior flags for the zeroth-index particle are stored in the zeroth index of the buffer. You can change particle types on the fly by writing new values to the buffer.

const b2Vec2 & b2ParticleGroup::GetPosition ( ) const
