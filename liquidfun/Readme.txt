Splash2D Version 1.0

Welcome to Splash2D!

Splash2D is a 2D physics engine for games.

Splash2D is an extension of Box2D, it adds a particle based fluid and soft body simulation to the rigid body functionality in Box2D.  Splash2D can be built for many different systems (Android, Windows, OS X, Linux), see Building.txt

For help with Splash2D, please visit http://www.stackoverflow.com.  You can post your questions there, tagged with "splash2d"

Please see Building.txt to learn how to build Splash2D and run the testbed.
